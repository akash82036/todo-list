# Todo List

A quick todo app built with React and Vite.  
Add, edit, delete and manage tasks with localStorage support.

## Features
- Add / Edit / Delete tasks
- Mark tasks as complete
- Filter tasks (All, Completed, Incomplete)
- Data stored in localStorage

## Tech Stack
- React  
- Vite  
- JavaScript  
- CSS

## About
Built as a small practice project to improve my React and UI skills.
